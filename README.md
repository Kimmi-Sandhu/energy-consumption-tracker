#Overview:
This project is a full-stack web application to track and manage energy consumption.It allow users to record and monitor their energy usage for different appliances.
This system provides a simple and user friendly interface for managing energy data.

#Features:

This application includes the following features:

User Registration

User Login

Add energy usage

Update energy records

Delete energy records

Edit energy records

View all entries

Track usage with date and units (kWh).
